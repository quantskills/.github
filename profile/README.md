# QUANTSKILLS 量化技能

QUANTSKILLS 是面向 AI Agent 时代的量化 Skill（能力包）、Agent（智能体）和 Strategy（策略）开放社区。

我们希望帮助量化开发者把交易经验、研究方法、因子模型和策略代码，变成可搜索、可安装、可验证、可传播的标准化资产。

> 把你的量化经验，变成 AI Agent 能使用的 Skill。

## 加入 QUANTSKILLS

开发者加入 QUANTSKILLS，不是为了免费贡献代码，而是为了把自己的量化能力变成公开资产。

你可以获得：

- 展示：进入 QUANTSKILLS GitHub 目录、官网、精选清单和社区推荐
- 信用：通过验证后获得 QS-Compatible、PandaData-Compatible、Backtest-Reproducible 等认证标识
- 分发：你的 Skill 未来可以被 AI Agent 搜索、安装和调用
- 合作机会：参与策略共创、内容共建、企业项目、认证服务和付费 Skill
- 个人品牌：从“我写过一个策略”变成“我发布了一个被社区收录和验证的量化能力包”

公开申请入口：

https://github.com/quantskills/join/issues/new?template=join-request.yml

申请 Issue 是公开的。请不要填写手机号、微信号、邮箱、身份证、密码、API Key 或任何私人账号信息。

## 我们收录什么？

QUANTSKILLS 关注三类资产：

- Skill：因子计算、数据清洗、策略审查、研报复现、报告生成等能力包
- Agent：研究复现、策略审查、内容生成、社区问答等智能体工作流
- Strategy：有代码、有回测证据、有风险说明、有适用边界的策略资产

## 如何参与？

1. 创建你的 Skill、Agent 或 Strategy
2. 上传到 GitHub
3. 提交到 QUANTSKILLS Registry
4. 获得展示、认证和传播

我们前期不强制统一模板。你可以自由创造，但如果希望被 QUANTSKILLS 收录，需要提交清晰的登记信息。

> 不用模板限制创造力，用登记和认证建立秩序。

## 认证体系

QUANTSKILLS 采用分层认证机制：

### Level 1: Listed 已登记

基本信息清楚，可以进入目录。

适合方法说明、提示词型 Skill、初期想法和教学示例。

### Level 2: Runnable 可运行

在已登记基础上，提供安装说明、示例输入、示例输出、可运行代码和依赖说明。

适合因子计算、数据处理、报告生成和简单策略脚本。

### Level 3: Verified 已验证

在可运行基础上，提供数据来源、无未来函数检查、回测证据、参数说明、风险说明和验证报告。

适合因子研究、策略研究、回测系统和可交易策略样例。

> 低门槛加入，高标准认证。

## AI Agent 如何发现？

QUANTSKILLS 不只给人看，也给 AI Agent 读取。

我们会逐步建设：

- llms.txt
- skills index
- agents index
- MCP 服务
- GitHub README、Topic 和 Release 规范

让 AI Agent 可以搜索、安装、调用和验证社区里的量化能力。

## 当前入口

组织创建初期，我们先保持轻量，不急于固定 Skill 的形态。

当前优先建设：

- quantskills/join：社区加入申请入口
- awesome-quant-skills：量化 Skill、Agent、Strategy 精选清单
- registry：Skill 登记目录

后续是否建设标准说明、示例项目、验证工具，会根据社区真实贡献和项目形态逐步决定。

我们更关心先让开发者能提交、能被看见、能被讨论，而不是一开始就用复杂模板限制创造力。

## 最终目标

如果你有一个量化方法、因子、策略或工具，你可以把它发布成 QUANTSKILLS Skill，让人看到，让 Agent 找到，让社区验证。
